# PotatoRNN
 [Python/Tensorflow] RNN으로 감자 가격 예측해보기
 
 # 프로젝트 설명
Python/Tensorflow, RNN을 이용하여 1996년 1월 ~ 2022년 1월의 감자 가격을 토대로 가격을 예측해보는 프로그램
